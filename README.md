# Security_Scanner
Using this tool, we can scan any website to verify basic security parameters using Python and Pytest.

## Project Structure
security_header.py: Contains the main program logic.
test_security_helper.py: Contains pytest logic to mock websites and test.

## Project URL:
https://github.com/apoorvgupta21/Security_Scanner

## Installation
Clone the repository:

```bash
python -m pip install -r requirements.txt
git clone https://github.com/apoorvgupta21/Security_Scanner.git
python security_helper.py
pytest test_security_helper.py -v
